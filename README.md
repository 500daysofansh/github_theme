## üöÄ Installation

### Option 1: Chrome/Edge/Brave (Recommended)
1. Install it via the [Chrome Web Store](Link-to-your-store-page).
2. Pin the extension and refresh GitHub.

### Option 2: Firefox
1. Install it via [Firefox Add-ons](Link-to-your-store-page).

### Option 3: Manual (For Developers)
1. Clone this repository: `git clone https://github.com/[YourUsername]/[RepoName].git`
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the folder you just cloned.
